# kalkulator
project kalkulator semester 1 ( tim: Syafiq Ahmad & Rio belly S)

project kalkulator ini memiliki fitur seperti perhitungan aritmatika pada umumnya, lalu juga ada perhitungan trigonometri, tak lupa dengan fitur darkmode-lightmode(fitur wajib disetiap website), dan terakhir fitur untuk memperbesar ukuran dari kalkulator.

project ini memiliki 3 file yaitu: 
1. calculator.html
2. calculator.css
3. calculator.js

1. calculator.html berisi struktur dasar dari kalkulator itu sendiri seperti class dan id dari button setting, history, dan kalkulator.
2. calculator css berfungsi untuk mengatur tampilan dari kalkulator itu sendiri seperti warna, ukuran, peletakan, animasi, dan sebagainya. lalu untuk pengaturan darkmode ada dibagian bawah dari codingan
3. calculator.js berisi tentang function dari kalkulator seperti function aritmatika, trigonometri dan sebagainya.
berikut adalah list function dari atas file kebawah
-Select all class and id
-Load preferences from localStorage
-Dark mode setting
-Toggle dark/lightmode
-hightlight button for calculator size button
-storage system for calculator size
-toggle for setting button
-toggle for menu trigonometri
-toggle for history
-refresh history
-display number
-keyboard system setting
-limit number in display
-function to math symbol
-backspace function
-update preview
-trigonometri calculate system
-calculation calculator system
